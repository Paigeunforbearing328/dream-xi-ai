# 赛场锦囊（Tips）

## 球场快捷操作

### @mention 点名传球

在聊天中使用 `@` 点名特定球员：

| 指令 | 效果 |
|------|------|
| `@leo` | 队长接球——架构设计、复杂推理 |
| `@andre` | 中场引擎——代码审查、安全分析 |
| `@flash` | 前锋出击——创意设计、快速原型 |
| `@wall` | 后卫接球——基础设施、兜底执行 |

不 @mention 任何人时，系统会根据任务类型自动路由给最合适的球员。

### 线程管理（战术板切换）

| 命令 | 说明 |
|------|------|
| `/new` | 开启新比赛（新线程） |
| `/threads` | 列出所有比赛 |
| `/use <id>` | 切换到指定比赛 |
| `/where` | 查看当前在哪场比赛 |

### 魔法关键词

| 关键词 | 触发效果 |
|--------|----------|
| "全力以赴" | 球队进入深度模式，全力处理当前任务 |
| "简单说" | 球队精简回复，只给关键信息 |
| "赛后总结" | 球队输出当前线程的完整工作总结 |
| "换人" | 将任务从一个球员手动交接给另一个 |

## 更衣室（Dugout）使用技巧

### 球员状态查看

在更衣室 → 阵容（Squad）标签页中，可以看到：

- 每名球员的当前状态（在线/离线/忙碌）
- 上下文预算和剩余容量
- 技能槽位和已加载的战术
- Token 使用统计

### 战术加载

球员会根据任务自动加载合适的战术（Skills）。你也可以手动指定：

```
@leo 使用 TDD 战术完成这个功能
```

## 语音陪练（Voice Companion）

### 开启方式

点击顶部导航栏的麦克风图标即可开启。

### 特性

- **每名球员独立音色** — 队长低沉稳重，前锋活泼快速
- **自动播放** — 回复自动排队播放，无需手动点击
- **按住说话** — 通过 ASR 语音识别输入指令

### 最佳使用场景

- 🏃 跑步时和球队讨论架构
- 🚗 通勤时做代码审查
- 🛋️ 躺着时 brainstorm 新功能

## 比赛模式（Game Modes）

### 狼人杀

- 7 人局，球员作为 AI 玩家参与
- 完整昼夜循环、投票、技能使用
- 裁判是确定性代码，不是 LLM
- 测试 A2A 通信和身份持久性的最佳方式

### 点球大战

- 实时像素对战 demo
- 测试回合制协调能力

## 效率技巧

1. **一线程一主题** — 不要在同一线程中混合多个任务
2. **让球员自己选位置** — 不确定该找谁时，直接描述任务，系统会路由
3. **善用赛后总结** — 长线程结束时让球队做总结，方便回顾
4. **定期清理战术板** — 归档已完成的线程，保持工作区整洁
5. **信任跨位置审查** — 中场审查队长代码，往往能发现盲点
