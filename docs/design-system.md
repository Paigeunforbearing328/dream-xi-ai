# 球队视觉识别系统（Design System）

Dream XI AI 的设计体系，确保全平台视觉一致性。

---

## 品牌色彩

### 主色调 — 冠军金 & 球场绿

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（冠军金） | `#D4AF37` | 世界杯奖杯金色，用于品牌标识、重要 CTA |
| 辅色（球场绿） | `#1B5E20` | 天然草坪绿，用于成功状态、确认操作 |
| 强调色（激情红） | `#D32F2F` | 红牌/错误状态、危险操作 |
| 中性色（夜空深蓝） | `#0D1B2A` | 深色背景、顶部导航 |
| 信息色（天际蓝） | `#1565C0` | 链接、信息提示 |

### 球员专属色

| 球员 | 色值 | 说明 |
|------|------|------|
| #10 Leo（队长） | `#7B1FA2` | 皇家紫 — 权威、战略 |
| #8 André（中场） | `#1565C0` | 稳重蓝 — 可靠、精准 |
| #9 Flash（前锋） | `#FF6F00` | 烈焰橙 — 能量、速度 |
| #4 Wall（后卫） | `#2E7D32` | 磐石绿 — 稳固、信赖 |

## 排版

### 字体

| 用途 | 字体 | 备选 |
|------|------|------|
| 标题 | **Inter** | -apple-system, BlinkMacSystemFont |
| 正文 | **Inter** | Segoe UI, Roboto |
| 代码 | **JetBrains Mono** | Fira Code, Consolas |
| 中文 | **思源黑体** | PingFang SC, Microsoft YaHei |

### 字号体系

| 级别 | 大小 | 行高 | 用途 |
|------|------|------|------|
| Display | 36px | 1.2 | 页面大标题 |
| H1 | 28px | 1.3 | 区块标题 |
| H2 | 22px | 1.4 | 子区块标题 |
| H3 | 18px | 1.4 | 卡片标题 |
| Body | 15px | 1.6 | 正文 |
| Small | 13px | 1.5 | 辅助文字 |
| Caption | 11px | 1.4 | 标签、时间戳 |

## 间距系统

基于 4px 网格：

| Token | 值 | 用途 |
|-------|-----|------|
| `--space-xs` | 4px | 紧凑元素间距 |
| `--space-sm` | 8px | 元素内间距 |
| `--space-md` | 16px | 标准间距 |
| `--space-lg` | 24px | 区块间距 |
| `--space-xl` | 32px | 大区块间距 |
| `--space-2xl` | 48px | 页面级间距 |

---

## 圆角

| Token | 值 | 用途 |
|-------|-----|------|
| `--radius-sm` | 4px | 按钮、输入框 |
| `--radius-md` | 8px | 卡片 |
| `--radius-lg` | 12px | 模态框 |
| `--radius-xl` | 16px | 大卡片 |
| `--radius-full` | 9999px | 头像、标签 |

## 球员头像系统

每名球员有独立的视觉标识：

| 球员 | 图标风格 | 视觉特征 |
|------|----------|----------|
| #10 Leo | 盾牌 + 皇冠 | 队长袖标，皇家紫色调 |
| #8 André | 齿轮 + 引擎 | 工程感，稳重蓝色调 |
| #9 Flash | 闪电 + 火焰 | 动态感，烈焰橙色调 |
| #4 Wall | 城墙 + 盾牌 | 防御感，磐石绿色调 |

## 动效规范

| 类型 | 时长 | 缓动函数 | 用途 |
|------|------|----------|------|
| 微交互 | 150ms | ease-out | 按钮 hover、toggle |
| 过渡 | 250ms | ease-in-out | 页面切换、面板展开 |
| 进入 | 300ms | ease-out | 模态框、通知 |
| 退出 | 200ms | ease-in | 关闭动画 |

## 暗色模式

Dream XI 默认使用暗色主题（夜场模式），契合深夜训练的氛围：

| 层级 | 色值 | 用途 |
|------|------|------|
| 背景-0（球场） | `#0D1B2A` | 页面底色 |
| 背景-1（更衣室） | `#1B2838` | 卡片背景 |
| 背景-2（替补席） | `#243447` | 悬浮元素 |
| 文字-主 | `#E8EAED` | 主要文字 |
| 文字-次 | `#9AA0A6` | 辅助文字 |
| 边框 | `#3C4858` | 分隔线 |

## 阴影与高程（Shadow & Elevation）

| 层级 | Token | CSS 值 | 用途 |
|------|-------|--------|------|
| 0 — 地面 | `--shadow-none` | `none` | 无阴影，贴地元素 |
| 1 — 草皮 | `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.3)` | 卡片默认状态 |
| 2 — 球场 | `--shadow-md` | `0 4px 12px rgba(0,0,0,0.4)` | 悬浮卡片、Dropdown |
| 3 — 看台 | `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.5)` | 模态框、浮层面板 |
| 4 — 包厢 | `--shadow-xl` | `0 16px 48px rgba(0,0,0,0.6)` | 全屏遮罩、命令面板 |
