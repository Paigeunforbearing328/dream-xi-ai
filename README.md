## ⚙️ How It Works

This software organizes AI agents into a team. Each agent handles a specific part of your work.

### Agent Roles

- Captain: Coordinates the team and manages the high-level plan.
- Playmaker: Breaks down complex requests into smaller executable steps.
- Striker: Executes specific tools and completes the primary work.
- Defender: Checks the output for quality and consistency.
- Keeper: Prevents errors and blocks risky actions.

The program guides you through assigning these roles. You choose the goal and the software arranges the team to reach that goal.

## 🛠 Using the App

When you launch the app, you see a clean dashboard. Follow these steps to start your first project.

1. Click the New Project button.
2. Enter your main task in the text box. Be specific about what you need.
3. Select your agent team in the setup panel. You can use a pre-built team or create your own.
4. Press the Start button.
5. Watch the dashboard as your agents communicate and work through the problem.
6. Review the final report once the agents finish.

## 🧩 Troubleshooting

Check these points if you run into trouble.

- Software won't open: Restart your computer and try again. Ensure no other instances are running in the background.
- Task hangs: Stop the current task and restart the agents. Check your internet connection.
- Slow performance: Close other open applications to free up memory for the agent team.
- Missing output: Click the Logs tab to see what the agents did during their shift.

## 🛡 Security and Safety

The software uses a keeper agent to maintain safety. This agent monitors all actions taken by the team. It blocks unauthorized file changes and manages web requests. You maintain control over these permissions in the settings panel. You can limit what external sites the team visits. You can also define which local folders the team can read.
